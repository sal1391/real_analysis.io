Proof style

How to write proofs in this guide

Use this structure for homework and exam answers. It follows the advanced-calculus style: state the claim, verify the definition or theorem assumptions, then conclude.

Claim:
State exactly what you are proving.

Proof:
Let epsilon > 0. State the goal.
Choose N in N such that ...
Then for all n >= N, calculate or estimate:
|x_n - L| = ... < epsilon.

Conclusion:
Therefore x_n converges to L.
Common proof mistake: Do not only compute. Explain why the computation proves the definition or verifies the theorem assumptions.
Definition

1. Convergent sequence

Real numbers

A sequence (xn) of real numbers converges to L in R if:

For every epsilon > 0, there exists N in N such that for all n >= N, |x_n - L| < epsilon.
Intuition: You give any small error tolerance epsilon. After some index N, every term must stay inside the band from L - epsilon to L + epsilon.

Metric spaces

In a metric space (X,d), a sequence (xn) converges to x in X if:

For every epsilon > 0, there exists N in N such that for all n >= N, d(x_n, x) < epsilon.

The only change is replacing absolute value distance |xn - x| with the metric d(xn,x).

Example: prove 1/n converges to 0

Claim: The sequence xn = 1/n converges to 0.

Proof: Let epsilon > 0. Choose N in N such that N > 1/epsilon. If n >= N, then

|1/n - 0| = 1/n <= 1/N < epsilon.

Conclusion: Therefore 1/n converges to 0.

Example: prove n/(n^2+1) converges to 0

For n in N, n2 + 1 > n2, so

0 < n/(n^2+1) < n/n^2 = 1/n.

Since 1/n goes to 0, the Squeeze Theorem gives n/(n2+1) goes to 0.

Non-example: (-1)^n does not converge

The even subsequence is constantly 1, so it converges to 1. The odd subsequence is constantly -1, so it converges to -1. If the original sequence converged, all subsequences would have the same limit. They do not, so (-1)n diverges.

Definition

2. Cauchy sequence

A sequence (xn) in R is Cauchy if:

For every epsilon > 0, there exists N in N such that for all n,k >= N, |x_n - x_k| < epsilon.

In a metric space, replace |xn - xk| with d(xn,xk).

Intuition: Convergent means the terms get close to a known limit. Cauchy means the terms get close to each other, even if you do not know the limit yet.
Example: 1/n is Cauchy

Let epsilon > 0. Choose N > 2/epsilon. If n,k >= N, then by the triangle inequality,

|1/n - 1/k| <= |1/n| + |1/k| <= 1/N + 1/N = 2/N < epsilon.

Therefore (1/n) is Cauchy.

Non-example: (-1)^n is not Cauchy

Take epsilon = 1. No matter how far out you start, you can find one even index and one odd index. The terms are 1 and -1, so the distance is 2, which is not less than 1. Therefore the sequence is not Cauchy.

Definition

3. Supremum and infimum

Supremum

s = sup A means s is the least upper bound of A.

  1. For every a in A, a <= s.
  2. If u is any upper bound of A, then s <= u.

Infimum

m = inf A means m is the greatest lower bound of A.

  1. For every a in A, m <= a.
  2. If l is any lower bound of A, then l <= m.
Intuition: Supremum is the tightest ceiling. Infimum is the tightest floor. They do not have to be elements of the set.
Example: sup(0,1) = 1

First, 1 is an upper bound because every x in (0,1) satisfies x <= 1. Second, no number u < 1 can be an upper bound because there is always a point between u and 1 that still lies in (0,1). Therefore the least upper bound is 1.

Example: inf{1/n : n in N} = 0

Every 1/n is positive, so 0 is a lower bound. If m > 0, choose n > 1/m. Then 1/n < m, so m cannot be a lower bound. Therefore the greatest lower bound is 0.

Definition

4. Limit superior and limit inferior

Let (xn) be a bounded sequence. Define the tail supremum and tail infimum:

a_n = sup{x_k : k >= n} b_n = inf{x_k : k >= n}

Then:

lim sup x_n = lim a_n lim inf x_n = lim b_n
Intuition: limsup is the eventual ceiling of the tails. liminf is the eventual floor of the tails. Ignore the first few terms and ask what high and low behavior keeps coming back forever.
SequencelimsupliminfReason
1/n00The tails shrink to 0.
(-1)^n1-1Both values appear forever.
0,1,2,0,1,2,...20Every tail still contains 0 and 2.
(-1)^n + 1/n1-1Even terms approach 1; odd terms approach -1.
1 if n is a square, 0 otherwise10Squares and non-squares both occur forever.
Convergence test: A bounded sequence converges if and only if limsup equals liminf. The common value is the limit.
Definition

5. Metric space and distance function

A metric space is a set X together with a function d: X x X → R satisfying, for all x,y,z in X:

RuleStatementMeaning
Positive definitenessd(x,y) >= 0 and d(x,y)=0 iff x=yDistance is never negative, and only the same point has distance 0 from itself.
Symmetryd(x,y)=d(y,x)Distance does not depend on direction.
Triangle inequalityd(x,z) <= d(x,y)+d(y,z)Going straight is no longer than taking a detour.

Examples

Standard metric on R

d(x,y)=|x-y|.

Discrete metric

d(x,y)=0 if x=y, and d(x,y)=1 if x is not y.

Euclidean metric on R^n

d(x,y)=sqrt((x_1-y_1)^2 + ... + (x_n-y_n)^2).

Arctan metric on R

d(x,y)=|arctan(x)-arctan(y)|.

Definition

6. Completeness

A metric space (X,d) is complete if every Cauchy sequence in X converges to a point inside X.

(X,d) complete means: Cauchy in X => converges in X.
Intuition: Complete spaces have no missing Cauchy limits. If a Cauchy sequence is trying to settle down, the place it settles must still be inside the space.
SpaceComplete?Reason
(R, usual metric)YesEvery real Cauchy sequence converges to a real number.
((0,1), usual metric)No1/n is Cauchy but converges to 0, which is not in (0,1).
([0,1], usual metric)YesA closed interval contains its limit points.
(Q, usual metric)NoA rational sequence can be Cauchy and converge to sqrt(2), which is not rational.
(R, arctan metric)NoThe sequence n is Cauchy under arctan distance but has no real limit in that metric.
(R, discrete metric)YesCauchy sequences are eventually constant, so they converge.
Theorem

7. Least Upper Bound Property of R

Precise statement: Every nonempty subset of R that is bounded above has a supremum in R.

Why it matters: This is the completeness property of the real numbers in order form. It says the real line has no holes.
Non-example in Q

The set A = {q in Q : q^2 < 2} is nonempty and bounded above in Q, but its natural least upper bound is sqrt(2), which is not in Q. This shows why Q does not have the same completeness property.

Theorem

8. Monotone Convergence Theorem

Precise statement: A monotone sequence of real numbers is bounded if and only if it converges. More specifically:

If (x_n) is increasing and bounded above, then x_n converges and lim x_n = sup{x_n : n in N}. If (x_n) is decreasing and bounded below, then x_n converges and lim x_n = inf{x_n : n in N}.

How to use it

  1. Show monotone: prove xn <= xn+1 or xn >= xn+1.
  2. Show bounded in the correct direction.
  3. Conclude the sequence converges by the Monotone Convergence Theorem.
  4. If asked for the limit, identify the supremum or infimum.
Common mistake: Do not say a sequence is monotone because it looks like it. Show it using xn+1-xn, xn+1/xn, or a direct comparison.
Theorem

9. Squeeze Theorem

Precise statement: If an <= xn <= bn for all sufficiently large n, and an → L and bn → L, then xn → L.

Intuition: If xn is trapped between two sequences that both close in on the same number, xn has no choice but to go there too.
Example: sin(n)/n converges to 0

Since -1 <= sin(n) <= 1, dividing by positive n gives:

-1/n <= sin(n)/n <= 1/n.

Both -1/n and 1/n converge to 0. Therefore sin(n)/n converges to 0 by Squeeze.

Theorem

10. Ratio Test for convergence of sequences

Let xn be nonzero eventually and suppose the following limit exists:

L = lim |x_(n+1) / x_n|.
  • If L < 1, then xn converges to 0.
  • If L > 1, then (xn) is unbounded and diverges.
  • If L = 1, the test gives no conclusion.
Example: x_n = 2^n/n!
|x_(n+1)/x_n| = [2^(n+1)/(n+1)!] [n!/2^n] = 2/(n+1) → 0.

Since 0 < 1, the Ratio Test says 2n/n! converges to 0.

When L = 1

The test is inconclusive. Examples: 1/n converges to 0, (-1)n diverges by oscillation, and n diverges to infinity. All have ratio limit 1 in the appropriate sense.

Theorem

11. Bolzano-Weierstrass Theorem

Sequence version: Every bounded sequence of real numbers has a convergent subsequence.

Intuition: Boundedness prevents escape to infinity. Even if the full sequence jumps around, some carefully chosen subsequence must settle down.
Example: (-1)^n

The sequence is bounded but not convergent. Still, it has convergent subsequences: the even subsequence is constantly 1, and the odd subsequence is constantly -1.

Set version

Every bounded infinite subset of R has at least one cluster point. This is the set version of the same idea: infinitely many points inside a bounded region must crowd somewhere.

12. How to prove a sequence converges

Direct epsilon proof

  1. Guess the limit L.
  2. Start with |xn-L|.
  3. Use scratch work to decide how large n must be.
  4. Choose N based on epsilon.
  5. Write the final proof in clean order.
Claim: x_n converges to L.

Proof: Let epsilon > 0. Choose N in N such that N > ____.
If n >= N, then
|x_n - L| = ____ <= ____ < epsilon.

Conclusion: Therefore x_n converges to L.

Theorem-based proof

Claim: The sequence converges.

Proof: We verify the assumptions of [theorem].
First, ...
Second, ...
Therefore, by [theorem], the sequence converges.

Conclusion: The theorem applies because all assumptions were checked.
Exam skill

13. How to compute limsup and liminf

Method A: tail sup/inf

  1. Write the n-tail: {xk : k >= n}.
  2. Find an = sup of that tail.
  3. Find bn = inf of that tail.
  4. Take lim an and lim bn.

Method B: subsequential limits

  1. Find the subsequences that capture the long-term high behavior.
  2. Find the subsequences that capture the long-term low behavior.
  3. The largest subsequential limit is limsup.
  4. The smallest subsequential limit is liminf.
Worked example: x_n = (-1)^n + 1/n

Even n: (-1)n=1, so xn=1+1/n → 1. Odd n: (-1)n=-1, so xn=-1+1/n → -1. Therefore:

limsup x_n = 1 liminf x_n = -1
Practice

Practice set with quick answers

1. Prove (3n+1)/(n+2) converges to 3.

|(3n+1)/(n+2)-3| = 5/(n+2). Choose N > 5/epsilon.

2. Show 1/sqrt(n) converges to 0.

Use a direct epsilon proof: choose N > 1/epsilon^2.

3. Find limsup and liminf of 0,1,2,0,1,2,...

limsup = 2 and liminf = 0 because every tail contains 0 and 2.

4. Is ((0,1), usual metric) complete?

No. The sequence 1/n is Cauchy but converges to 0, which is outside (0,1).

5. State Bolzano-Weierstrass.

Every bounded sequence of real numbers has a convergent subsequence.
